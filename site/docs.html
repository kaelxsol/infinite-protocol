Using the API

The Infinite proxy sits between you and major AI providers. You make a standard request, we handle the auth and billing via the treasury.

curl https://api.infinite.sh/v1/chat \ -H "Authorization: Bearer YOUR_INFINITE_KEY" \ -d '{ "model": "claude-3-5-sonnet", "messages": [{"role": "user", "content": "Analyze this ticker..."}] }'

We currently support mapped endpoints for:

  • Anthropic (Claude 3.5 Sonnet & Haiku)
  • Google (Gemini 1.5 Pro & Flash)
  • X API (Search, Trends, Posts)
  • Flux (Image Generation)

Chat & Agents

Infinite provides unified access to the world's best LLMs. You don't need separate subscriptions. Your API key routes requests intelligently to the provider you specify.

Claude 3.5 Sonnet

Best for coding, complex reasoning, and nuance. The default model for most agents.

Gemini 1.5 Pro

Massive context window (2M tokens). Perfect for analyzing entire codebases or large documents.

Trading Tools

We've built specialized agents for Solana traders. These run on our infrastructure but are controlled by you.

  • Wallet Monitor
    Tracks specific wallets for movement. Alerts you on Discord or Telegram when a "smart money" wallet buys.
  • Trend Scanner
    Uses X API to scan for ticker mentions. Correlates social volume with price action to find early breakouts.
  • Sentiment Analyzer
    Feeds 100+ recent tweets about a token into Claude to get a qualitative analysis of community sentiment (bullish/bearish/scam).
