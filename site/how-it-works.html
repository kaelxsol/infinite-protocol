02 — The Mechanism

Creator fees fund everything.

pump.fun — the platform where $INFINITE is deployed — charges a creator fee on every trade. This fee ranges from 0.05% to 0.95% based on market cap (smaller tokens generate higher fees). Instead of pocketing this revenue, INFINITE routes it to an API treasury.

1

Someone buys or sells $INFINITE

Every trade on pump.fun generates a creator fee — between 0.05% and 0.95% of the trade value, paid in SOL.

2

Fees split to the treasury

Creator fees are automatically split: 50% goes to the API treasury wallet, 40% to development, 10% to community. All on-chain, all auditable.

3

Treasury funds AI access

The treasury converts SOL to API credits with Anthropic (Claude) and Google (Gemini). An autonomous agent manages this conversion and adjusts rate limits based on available runway.

4

Holders get API keys

Connect your wallet, verify your token balance, and receive an API key. Every call you make routes through our proxy — which checks your balance, enforces rate limits, and forwards to the real APIs using our master keys.

5

You use AI for $0

Make API calls to Claude and Gemini. Use pre-built trading agents. Access the AI chat. Your cost: nothing. The treasury covers it.

Drop in your key and build.

INFINITE gives you raw API keys that work with standard Claude and Gemini SDKs. Just point your base URL at our proxy and use your INFINITE key instead of an Anthropic or Google key. No billing accounts. No credit cards.

07 — Security & Trust

Built for verification, not faith.

Wallet-Bound Keys

Every API key is tied to a specific Solana wallet. One key per wallet. You can't share keys without sharing your wallet.

Live Balance Checks

Your token balance is re-verified via Helius on every request (with a 5-minute cache). Sell below the threshold? Your key auto-deactivates.

Proxy Architecture

You never see our master API keys. Your inf_ keys only work against our proxy. We control rate limits, model access, and abuse detection.

On-Chain Treasury

The API treasury wallet is public. Anyone can verify the balance, track fee inflows, and audit how funds are used. Full transparency.

08 — FAQ

Common questions.

What happens if I sell some tokens and drop below my tier?
Your access automatically adjusts to match your new balance. If you drop below the minimum threshold (10K tokens), your API key deactivates until you hold enough again. No penalty — just re-buy and your key reactivates.
What if trading volume drops to zero?
The treasury maintains a runway buffer. Even if volume drops temporarily, existing treasury funds continue to cover API calls. The treasury agent dynamically adjusts rate limits based on available runway to ensure sustainable access.
Can someone just hold the minimum and make millions of API calls?
No. Each tier has a daily call limit. Signal (10K tokens) gets 1,000 calls/day. Operator (100K) gets 10,000. Only Architect (1M+) gets unlimited, and even then, there are rate limits to prevent abuse.
Why not just charge a subscription instead?
Because subscriptions extract money from users. INFINITE lets you own your access. The token you buy can appreciate in value while simultaneously giving you free AI access. It's investment + utility in one.
Which AI models are available?
Currently: Claude Sonnet, Claude Opus, Gemini Flash, and Gemini Pro — depending on your tier. More models will be added as the treasury grows and new providers become available.
Is the treasury auditable?
Yes. The treasury wallet address is public. Every SOL inflow from creator fees and every outflow for API credits is visible on-chain. We'll also publish a live dashboard showing treasury balance, daily API spend, and runway estimates.
